# Snippets Catcher plugin

## Motivation

The other day I though - "Why am I still installing my CSS snippets manually? I can make it easier!"

## How to use it

1. Install & enable Plugin;
2. Go to `Settings` -> `Snippets Catcher`;
3. Download any amount of snippets you like;
4. Activate newly downloaded snippets at `Settings` -> `Appearance`.

> []
> Available snippets are being pulled from [opa-oz/obsidian-snippets](https://github.com/opa-oz/obsidian-snippets). Feel free to contribute, or switch to your own repository
